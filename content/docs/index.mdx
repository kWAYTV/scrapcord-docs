# Welcome to Scrapcord

Scrapcord is a powerful Discord bot designed to manage and automate your Rust game servers through Pterodactyl panel integration. With features like automated wipes, server management, and configuration handling, Scrapcord makes server administration easier than ever.

## Key Features

<Cards>
  <Card title="Server Management" href="/docs/features/server-management">
    Add and manage multiple Rust servers with ease
  </Card>
  <Card title="Automated Wipes" href="/docs/features/automated-wipes">
    Schedule and automate your server wipes with flexible configurations
  </Card>
  <Card title="Configuration System" href="/docs/features/configuration">
    Powerful YAML-based configuration system for each server
  </Card>
  <Card title="Discord Integration" href="/docs/features/discord-integration">
    Full Discord slash command support and role-based permissions
  </Card>
</Cards>
